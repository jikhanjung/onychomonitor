# 카메라 촬영 스크립트 구성

**날짜**: 2026-03-23 ~ 2026-03-24

## 작업 내용

### 1. 촬영 스크립트 작성 (`capture.sh`)

- `rpicam-still`을 사용하여 사진 촬영
- 파일명 형식: `YYYYMMDD-HHMMSS.jpg`
- 촬영 흐름: `/tmp` (tmpfs)에 임시 저장 → `/nfs/share/onychomonitor/pictures`로 이동

### 2. 로깅 추가

- 로그 파일: `/srv/onychomonitor/capture.log`
- 성공 시 `[OK]`, NFS 미마운트 시 `[WARN]`, 오류 시 `[ERROR]` 로 구분하여 기록

### 3. NFS 장애 대비 로컬 백업

- NFS 미마운트 또는 이동 실패 시 `/srv/onychomonitor/tmp_pictures/`에 로컬 저장
- NFS 복구 후 다음 촬영 시 밀린 사진 일괄 전송 (flush)

### 4. cron 등록

- 1분 간격으로 촬영 실행
- `* * * * * /srv/onychomonitor/capture.sh`

## 파일 구조

```
/srv/onychomonitor/
├── capture.sh          # 운영용 스크립트
├── capture.log         # 촬영 로그
└── tmp_pictures/       # NFS 장애 시 임시 저장소

/home/paleoadmin/projects/onychomonitor/
├── capture.sh          # 개발용 원본
├── pictures -> /nfs/share/onychomonitor/pictures
└── devlog/
```
