# 사진 저장 날짜별 디렉토리 구조 적용

**날짜**: 2026-04-13

## 작업 내용

### 1. 날짜별 하위 디렉토리 생성

- 사진 저장 경로를 `pictures/YYYYMMDD/`로 변경
- 예: `pictures/20260413/20260413-143000.jpg`
- 촬영 시 해당 날짜 디렉토리가 없으면 `mkdir -p`로 자동 생성

### 2. 밀린 사진 flush 시 날짜 분류

- `tmp_pictures`에 쌓인 사진을 NFS로 전송할 때 파일명에서 날짜(`YYYYMMDD`) 추출
- 각 사진을 촬영 날짜에 맞는 디렉토리로 분류하여 전송

### 3. 배포

- `cp capture.sh /srv/onychomonitor/capture.sh`로 운영 디렉토리에 배포
