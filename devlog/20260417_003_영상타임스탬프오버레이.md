# 영상 생성 시 타임스탬프 오버레이 추가

**날짜**: 2026-04-17

## 작업 내용

### 1. 타임스탬프 표시 기능 추가

- `make_video.sh`에서 영상 생성 시 각 프레임의 촬영 시각을 화면 우측 하단에 표시
- 파일명(`YYYYMMDD-HHMMSS.jpg`)에서 시각 추출하여 `YYYY-MM-DD H:MMam/pm` 형식으로 변환
- 예: `20260416-143000.jpg` → `2026-04-16 2:30pm`

### 2. 구현 방식

- ASS 자막 파일을 임시 생성하여 ffmpeg `ass` 필터로 합성
- 흰색 텍스트 + 검은 외곽선/그림자로 배경에 관계없이 가독성 확보
- 프레임별 정확한 타임코드 계산으로 각 사진에 맞는 시각 표시
- 임시 파일은 스크립트 종료 시 자동 정리 (`trap`)

### 3. 의존성

- `libass` 필요 (paleoserver에 설치 확인 완료)
